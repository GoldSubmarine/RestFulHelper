## Usage üë£

- Search everywhere (```Shift``` twice)
- Keyboard shortcuts:
  - MacOS: ```Cmd + Back slash```
  - Linux: ```Ctrl + Back slash```
  - Windows: ```Ctrl + Back slash```
- Navigate (menu bar) -> Request mapping
